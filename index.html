Enhanced Custom Fields

These custom fields collect information and KPIs on issues in the background, so you don’t have to do all of the work. Just sit back and analyse the data on user-friendly screens. Button to JEP custom fields page (actual linking will be done with coding in plugin).

Duration Custom Fields

Helping you calculate the time spent on particular statuses within an issue. Working calendars are also used for this, making calculations even more relevant to your team.

Find Out More
Date Custom Fields

Allowing you to see certain milestones within an issue, such as when it first entered into the “WAITING RESPONSE” status.

Find Out More
User/Group Custom Fields

Showing you exactly which users undertook particular tasks within an issue which is really useful if you have multiple teams working on an issue at different stages. The plugin also offers two filterable user select custom fields to enhance Jira’s user/group picker custom fields (which display all users and groups without any filtering options).

Find Out More
Counter Custom Fields

Calculating how many times a particular action occurred within an issue. Our customers love to use this within their customer service portals to calculate how many times an issue was reopened, as this has an impact on first resolution.

Find Out More
Enhanced Workflows

An out of the box solution that enhances your Jira workflow conditions, validators and post functions without the need for you to spend time coding, scripting or maintaining these functions.

Enhancer Plugin for Jira gives you access to the most sought after conditions, valitators and post functions with zero effort. Boost Jira’s default workflow implementation and get the most out of your workflows.

See Workflows Page
Workflow Conditions
  • User Group and/or Project Role
  • User Custom Field
  • Previous Status
  • Parent Issue JQL
  • User Non-Reporter
  • Date Comparison
Workflow Validators
  • Field Has Been Set
  • User Project Role
  • Parent Issue Status
  • Parent Issue JQL
  • Subtask JQL
  • Date Comparison
Workflow Post Functions
  • Assign Issue to User in Custom Field
  • Assign Issue to Specific User Role (incl. Based on Issue Field)
  • Notify When Field Set in Transition
  • Update Linked Issue Custom Field/Field Value With Current Issue
  • Add Subtask User(s) to Parent Issue Watcher List
Enhanced Reports & Gadgets

Custom fields offer great KPIs within issues but it can be difficult to see the bigger picture when managing a project with multiple workflows. This is where Enhancer Plugin’s reports and gadgets come in…

Gadgets give you quick and live insights such as what the average response time in the last week is, so you can manage all of your projects with ease.

Link to Jira dashboards page Find Out More
Average Time Between Bar Chart

This handy gadget does exactly what it says on the tin, it displays the average amount of time (either weekly or monthly) between transitions. Use this to check the amount of time between In Progress - Resolved status for example.

Find Out More
User Performance Gadget

Want to see the performance of your team at one glance? Then use this gadget to display valuable metrics such as the resolutions count, original estimation vs resolution time, reopenings, and linked issues per user.

Find Out More
Number Custom Field Analyzer

This colorful gadget displays the analyzed results of the selected number custom field and groups them by any field within that issue. Our customers love this trusty gadget because it does all of the hard work for you.

Find Out More
Resolutions vs Reopenings Gadget

This gadget is another customer favorite. You can use it to display the resolution counts versus the reopening counts for each user. A simple but effective gadget that you’ll wish that you’d discovered sooner.
